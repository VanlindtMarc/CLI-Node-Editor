# CLI Node Editor

Ce logiciel a pour but de générer automatiquement des fichiers BATCH, BASH et POWERSHELL complexes faisant appels à plusieurs logiciels en ligne de commande afin d'automatiser un flux sur un ou plusieurs fichiers.

**Attention : ** Ce logiciel est toujours en cours de création et il peut planter lorsque l'on modifie des noeuds déjà en place. Pensez à sauvegarder votre flux couramment. 

## Entrée

Trois modes sont prévus : 

1. Chaque fichier est traité individuellement
2. Tous les fichiers sont traité par une seule commande
3. Un certain nombre de fichier est demandé, chacun d'un certain type, et chacun sera sélectionné par son extension pour la suite du flux
